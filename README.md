# 🎸 Guitar Simulator

Симулятор гитары с визуализацией аккордов на грифе и воспроизведением звуков настоящей гитары.

## 📋 Возможности

- **14 аккордов**: C, D, E, F, G, A, H и их минорные версии (Cm, Dm, Em, Fm, Gm, Am, Hm)
- **Визуализация на грифе**: отображение зажатых ладов для выбранного аккорда
- **Два способа звукоизвлечения**:
  - **Перебор** — извлечение отдельных струн (клавиши 1-6)
  - **Бой** — удар по всем струнам вверх/вниз (PageUp/PageDown)
- **Записанные сэмплы** — используются реальные звуки гитары

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| `C`, `D`, `E`, `F`, `G`, `A`, `H` | Выбрать аккорд |
| `M` | Переключить минор/мажор |
| `1` – `6` | Извлечь струну |
| `PageUp` | Удар по струнам вверх |
| `PageDown` | Удар по струнам вниз |
| `F1` | Открыть инструкцию |

## 📦 Установка

```bash
# Установить зависимости
pip install -r requirements.txt

# Запустить приложение
python main.py
```

## 🏗️ Структура проекта

```
guitar/
├── main.py              # Основной код приложения
├── requirements.txt     # Зависимости
├── database/            # База данных с аккордами и звуками
├── audio/               # Аудиофайлы (сэмплы струн и боёв)
└── main.exe             # Готовый исполняемый файл (Windows)
```

## 🛠️ Технологии

- **PyQt5** — графический интерфейс
- **SQLite** — хранение данных об аккордах и звуках
- **playsound** — воспроизведение аудио

## 📝 Примечание

Для корректной работы приложения необходима база данных `database/guitar_base.db` и аудиофайлы в папке `audio/`.
